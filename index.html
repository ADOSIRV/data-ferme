ℹ️ Comment fonctionne l'authentification Tuffigo Rapidex ?
L'API REST utilise un token statique transmis dans le header HTTP :
Authorization: token <votre_token>
Le login / mot de passe donnent accès au portail web Tuffigo et sont stockés ici pour référence et pour les workflows N8N.
